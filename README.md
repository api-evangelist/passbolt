# Passbolt

Passbolt is an open source password manager for teams. This repository tracks the Passbolt API along with related developer resources.

## APIs

### Passbolt API

Low-level reference covering authentication, resources, folders, users, groups, sharing, comments, metadata keys, and healthcheck. JWT is the preferred authentication method; legacy GPGAuth is also supported.

- **Documentation:** https://www.passbolt.com/docs/api/
